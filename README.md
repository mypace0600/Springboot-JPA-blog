목적

Spring Security, JPA를 연습하기 위해서 만들어본 블로그 입니다.
<br/>
<hr />
기술스택

JAVA 11,
Spring Boot,
Spring security,
JPA,
Mysql,
Javascript,
Bootstrap

<br/>
<hr />
기능설명

 계정)
 
  -로그인,로그아웃
  
  -카카오로그인(OAuth2)
  
  -회원가입, 회원탈퇴
  
  -회원정보 수정
  
  -관리자권한 신청
  

 게시판)
 
  -게시글 CRUD
  
  -댓글CRUD
  
  -게시글 비공개 설정
  
  -조회수 중복 방지(Redis)
  

 관리자)
 
  -관리자권한 부여,해제
  
  -계정삭제
  
  -사용자별 게시글 목록
  
  -사용자게시글 RUD
